# To Be Free Growth Plan

## Measurement

1. Add the site to Google Search Console and Bing Webmaster Tools.
2. Submit `https://tobefree.app/sitemap-index.xml`.
3. In Cloudflare Pages, set `PUBLIC_CF_WEB_ANALYTICS_TOKEN` to the Web Analytics token.
4. Review weekly:
   - Search queries with high impressions and low CTR.
   - Pages ranking in positions 8-30.
   - Pages with traffic but low outbound clicks.

Improve self-made tool acquisition:

## Page Template

For every tool page, cover these points:

- What problem it solves.
- Who should use it.
- Whether it is free, open source, ad-free, offline, and cross-platform.
- Where it is better than the common paid or ad-supported alternatives.
- Its limits or tradeoffs.
- Official website, download page, and source code where available.

## Distribution

- Share category pages in communities where the problem is active, not generic launch sites.
- Turn each collection page into a short post for GitHub, X, Reddit, V2EX, Hacker News or relevant forums.
- Ask maintainers of listed open-source tools to link back when the page is useful and accurate.
